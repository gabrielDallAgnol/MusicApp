# MusicApp

Aplicativo organiza os 30 artistas mais tocado por genero, com base na API da Last.fm, te permitindo perquisar os artitas pelo nome.

### Prerequisitos

Voce vai precisar ter um MACOS e ter instalado:

Xcode e CocoaPods

3 - execute 'pod install';
